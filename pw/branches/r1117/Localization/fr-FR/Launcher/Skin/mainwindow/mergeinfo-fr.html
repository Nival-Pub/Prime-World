Règles de liaison des comptes

Lorsque vous ajoutez un nouveau mode d'identification, le cas suivant peut se produire : ce mode d'identification est déjà lié à un compte. Cela se produit quand les deux comptes ont été utilisés pour entrer dans le jeu et créer un personnage/château. Dans ce cas, vous pouvez relier ces deux comptes.

Important : il est possible de relier les réseaux sociaux au compte créé avec une adresse e-mail à partir de la fenêtre de chargement, mais l'inverse est impossible.

Important : vous ne pouvez relier les réseaux sociaux au compte créé avec une adresse e-mail que si vous avez confirmé votre enregistrement.

Important : en ajoutant de nouveaux modes d'identification, vous pouvez recevoir jusqu'à deux compétences.

Important : en changeant de compte principal, vous modifiez l'adresse e-mail de paiement. Lorsque vous achetez de l'or, veillez à toujours utiliser l'adresse e-mail de paiement indiquée sur la page « Approvisionner mon compte » de votre espace personnel sur le site du jeu.

La liaison s'effectue de la manière suivante :

  1. Le château et le personnage de l'un des comptes seront supprimés après la liaison. Vous pouvez choisir lequel de vos comptes vous voulez conserver. Celui-ci deviendra votre compte principal.
  2. Le contenu de votre autre compte sera entièrement supprimé, sauf l'or, qui sera transféré sur votre compte principal. Vous recevrez également la quantité d'or dépensée pour les héros et pour l'achat de bâtiments dans le château (les maisons des animaux).
  3. Tous les autres paramètres du compte principal (sexe, adresse e-mail de paiement, compte du forum etc.) seront conservés.

Si le mauvais compte a été supprimé par erreur, nous pouvons tenter de le rétablir. Les chances de réussite pour le rétablir dépendent directement de la rapidité avec laquelle vous contacterez le support technique. Afin d'éviter ce type de situation, nous vous prions d'être attentif lors du choix du compte principal.
